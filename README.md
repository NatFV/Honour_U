“Honour U” es una plataforma colaborativa diseñada para crear homenajes personalizados a personas queridas. Su objetivo principal es facilitar la recopilación de mensajes, fotografías, vídeos y audios procedentes de diferentes participantes, transformando todo ese material en un recuerdo único: un libro físico o un libro digital. La herramienta está pensada especialmente para usuarios con poco tiempo y con capacidad organizativa, que desean preparar un regalo significativo para ocasiones como cumpleaños, jubilaciones, bodas o cualquier celebración especial.
La aplicación responde a varias necesidades habituales en este tipo de proyectos:
-	Centraliza en un único espacio la recolección de contenido en distintos formatos, evitando la dispersión y la pérdida de información.
-	Simplifica la participación de colaboradores, reduciendo el riesgo de que aportaciones se pierdan por sistemas poco intuitivos o procesos demasiado largos.
-	Ofrece un diseño estructurado y automatizado para generar el libro final, eliminando la carga de maquetación manual y ahorrando tiempo al organizador.

El desarrollo de Honour U se ha realizado con tecnologías modernas y robustas:
-	IntelliJ IDEA como entorno de desarrollo.
-	Java 17 y Spring Boot para la construcción del backend y los servicios REST.
-	Maven/Gradle como gestores de dependencias.
-	MySQL para la gestión y almacenamiento de datos.
-	Thymeleaf para la generación dinámica de páginas web.
-	Postman para la verificación y prueba de los endpoints de la API.
El resultado es una aplicación que permite crear un proyecto de homenaje, compartir un enlace con los participantes para que añadan sus aportaciones y, finalmente, ordenar y personalizar el contenido desde un panel de control. El organizador puede configurar la portada y generar un libro impreso o un PDF, dando forma a un recuerdo emotivo, colaborativo y profesional.
